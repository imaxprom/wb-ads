# Нижняя панель — детализация товара

## Общее
- Экран делится горизонтально на 2 части: верх (таблица товаров) + низ (детализация)
- Разделитель — перетаскиваемая горизонтальная линия (resize вверх/вниз)
- Нижняя панель появляется при клике на строку товара в верхней таблице
- Слева — фото товара (большое), название, описание
- Справа — табы: **По дням** | Остатки | Каталоги | Запросы

## Таб "По дням" — столбцы (в порядке как на скриншоте)

| # | Заголовок | Описание (тултип) | Источник данных |
|---|-----------|-------------------|-----------------|
| 1 | Дата | Дата по Мск | date |
| 2 | 👁 общ. | Увидели карточку — сколько раз покупатели видели карточку в выдаче каталога, поиска и рекомендаций | sales_funnel_daily.open_card_count |
| 3 | CTR* | Конверсия общих показов в переходы (клики), %. ~ — слишком мало показов. Условно, т.к. не все переходы могут идти из выдачи | расчёт: clicks/views*100 |
| 4 | 👁→ общ. | Общее кол-во переходов в карточку товара (и с органики, и с рекламы) | sales_funnel_daily.open_card_count (или отдельное поле) |
| 5 | 🛒→ % | Конверсия общих переходов в корзины | add_to_cart_conversion |
| 6 | 🛒 общ. | Общее кол-во товаров помещённых в корзину | sales_funnel_daily.add_to_cart_count |
| 7 | 📦→ % | Конверсия корзин в заказы. ## — сбой/запаздывание статистики корзин | cart_to_order_conversion |
| 8 | Ср. цена | Средняя цена товара по заказам покупателей. Цена поставки (без скидки WB "СПП") | orders_sum / orders_count |
| 9 | ДРРз | Доля рекламных расходов. # % — нет заказов. ~ — < 0.1% | ad_spend / orders_sum * 100 |
| 10 | Счет | Сумма затрат (начислений) на рекламу. ~ — менее 1₽ | campaign_stats_by_nm.sum (или daily) |
| 11 | Сум. заказы | Сумма всех заказов товара по ценам поставки | sales_funnel_daily.orders_sum |
| 12 | 👁 рекл. | Показы рекламы | campaign_stats_by_nm.views |
| 13 | CTR | Конверсия показов рекламы в клики, % | campaign_stats_by_nm.ctr |
| 14 | 🖱 клики рекл. | Кол-во кликов с рекламы × Средняя цена клика | clicks × cpc |
| 15 | CPM | Средняя цена 1000 рекл.показов | campaign_stats_by_nm.cpm или расчёт |
| 16 | 🛒 рекл. | Добавления в корзину рекламируемого товара. +? — есть ещё от рекламы других товаров | campaign_stats_by_nm.atbs |
| 17 | 🛒 xP | Рекламная себестоимость корзин. # — расход есть, но корзин нет | ad_spend / atbs |
| 18 | 📦 рекл. | Заказы с рекламы этого товара. +? — есть ещё от рекламы других товаров | campaign_stats_by_nm.orders |
| 19 | 📦 xP | Рекламная себестоимость заказов. # — расход есть, но заказов нет | ad_spend / orders |
| 20 | +🛒 | Ассоциированные корзины других товаров от рекламы этого | заглушка |
| 21 | +📦 | Ассоциированные заказы других товаров от рекламы этого | заглушка |
| 22 | Куп | Выкуплено заказанных товаров, шт | sales_funnel_daily.buyouts_count |
| 23 | Отм | Сколько заказов отменили, шт | sales_funnel_daily.cancel_count |
| 24 | ~?~ | Сколько заказов в пути, шт | расчёт: orders - buyouts - cancels |
| 25 | % вык | Процент выкупа. ~?~ — данных недостаточно | sales_funnel_daily.buyout_percent |
| 26 | 🛒→📦 % | Конверсия рекламных кликов в корзины | расчёт |
